6.1 Что такое файл ввода?
    - Файл ввода, это файл, в который что-то записывает программа.

6.2 Что такое файл вывода?
    - Файл вывод, это файл, из которого программа что-то читает.

6.3 Какие 3 шага программа должна сделать, когда она использует файл?
    - 1. Открыть файл
    - 2. Поработать с файлом
    - 3. Закрыть файл

6.4 Какие бывают два типа файлов? Разница
    - Текстовый и Бинарные. Текстовые предназначены для человека, т.е. содержимое таких файлов может прочитатать
    человек и различные текстовые процессоры. Бинарный файлы предназначены для других программ, содержимое таких файлов
    нельзя прочитать без специальных программных средств.

6.5 Два типа доступа к файлу?
    - 1. Последовательный: что-бы добраться до конца файла, нужно прочитать все символы\байты, которые находятся перед
    этим концом*.
    - 2. Прямой доступ: можно читать от куда угодно файл, за конечное время.

6.7 Что происходит при попытки открыть существующий файл с режимом w?
    - Содержимое файла перетирается.

6.8 Какова задача отрытия файла?
    - Связать файл с объектом в программе.

6.9 Какова задача закрытия файла?
    - Разорвать связь между объектом в программе и файлом, на который ссылается этот объект.

6.10 Что такое позиция считывания файла? Где она при открытии файла?
    - 1. Указатель на позицию в файле откуда нужно читать данные. При чтении данных позиция сдвигается на то место, где была окончено чтение.
    - 2. В начале файла.

6.11 Как записать новый данные в файл, при этом сохранить содержимое? Куда добавятся данные в файле?
    - 1. Режим "A" - Append = добавить.
    - 2. В конец файла.
